Regular sudoku project 

In this Project We made a 9X9 sudoku Board.
and an Interface to play it + Solver.
See "Instructions" for supported commands and how to use.

In this project we Use the Backtracking algorithm for the solver,
see Solver Class for Implementaion.

Main: Responsible for recieving commands from the User.
MainAux: Responsible for the printing of the board.
Parser: Responsible for parsing the commands from the User
Game: Responsible for executing the User Commands.
Solver: Responsible for the Backtracking algorithm for sloving the Board.

To Run, first run the makefile and then the executable created
